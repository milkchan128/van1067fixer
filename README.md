# VALORANT van 1067 error fixer

## 사용법:

1. 릴리즈에 있는 exe파일을 다운로드 한다.
2. 해당 파일을 실행하고 Fix and Reboot 버튼을 클릭한다.

## 주의사항:

- 버튼을 클릭하는 즉시 오류해결 명령줄이 실행되며 5초후 재부팅됩니다. 만약 잘못눌렀다면 아래 Cancel Reboot 버튼을 클릭하세요.
